# ⚽ Football Player Market Value Prediction

## Project Overview

This project aims to predict the market values of professional football players using Transfermarkt data and Machine Learning techniques. The dataset includes player characteristics, club information, and performance statistics such as goals, assists, minutes played, and disciplinary records. The objective is to estimate a player's market value based on these features.

During the project, data preprocessing, feature engineering, missing value handling, categorical variable encoding, outlier analysis, and model comparison steps were applied. Several regression algorithms were tested and evaluated.

## Dataset

The project uses Transfermarkt football player data containing:

- Player information
- Position and sub-position
- Preferred foot
- Height
- Nationality
- Club information
- Goals
- Assists
- Minutes played
- Yellow cards
- Red cards
- Historical market values

## Data Preprocessing

The following preprocessing steps were performed:

- Merging multiple Transfermarkt datasets
- Missing value handling
- Feature selection
- Aggregation of player performance statistics
- One-Hot Encoding for categorical variables
- Outlier analysis
- Logarithmic transformation of target variable

## Models Used

- Linear Regression
- Random Forest Regressor
- XGBoost Regressor

## Model Performance

### Before Log Transformation

| Model | R² Score |
|---------|---------:|
| Linear Regression | 0.32 |
| Random Forest Regressor | 0.59 |
| XGBoost Regressor | 0.48 |

### After Log Transformation

| Model | R² Score |
|---------|---------:|
| Linear Regression | 0.37 |
| Random Forest Regressor | 0.60 |
| XGBoost Regressor | 0.51 |

Random Forest Regressor achieved the best performance and was selected as the final model.

## Conclusion

This project demonstrates how Machine Learning techniques can be applied to sports analytics and player valuation. By combining player attributes, club information, and performance statistics, the model can explain approximately 60% of the variation in player market values. Among the tested algorithms, Random Forest provided the most reliable results and was selected as the final prediction model.
